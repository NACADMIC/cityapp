## 🔧 알리고 SMS 설정 방법

### 1단계: 알리고 가입

1. **알리고 회원가입**
   - https://www.aligo.in 접속
   - 회원가입 및 사업자 정보 등록

2. **충전**
   - 알리고 대시보드에서 충전
   - 최소 충전 금액 확인

### 2단계: API 키 발급

1. **API 관리**
   - 알리고 대시보드 → API 관리
   - API Key 확인
   - User ID 확인

2. **발신번호 등록**
   - 발신번호 관리 → 발신번호 등록
   - 사업자 등록증 필요 (인증 필요)

### 3단계: 환경 변수 설정

**Railway 환경 변수 추가:**

```
SMS_SERVICE=aligo
SMS_API_KEY=your_aligo_api_key
SMS_API_SECRET=your_aligo_user_id
SMS_SENDER=01012345678
```

**테스트 모드 (실제 발송 안 함):**
```
SMS_TEST_MODE=true
```

## 💰 예상 비용

**월 주문 100건 기준:**
- SMS 3회/주문 = 300건/일 = 9,000건/월
- 비용: 약 13.5만원/월 (건당 15원 기준)
- 대량 할인 시 더 저렴 가능

**비용 절감 팁:**
- 필수 알림만 발송 (예: 배달 완료만)
- 대량 할인 요금제 사용
- 월별 발송량에 따라 할인율 확인
