# ⚡ 빠른 배포 가이드

## 🚀 3단계로 배포하기

### 1단계: Git 커밋 및 푸시
```bash
cd backend
git add .
git commit -m "모든 기능 구현 완료"
git push origin main
```

### 2단계: Railway 배포

**Railway 대시보드:**
1. 프로젝트 선택
2. Settings → Source → Connect GitHub
3. 저장소 선택
4. 자동 배포 활성화

### 3단계: 환경 변수 설정

**Railway Variables 탭에서 추가:**
```
DATABASE_URL=postgresql://... (자동 생성됨)
KAKAO_TEST_MODE=true (테스트용)
```

---

## ✅ 완료!

배포가 완료되면 Railway URL로 접속하세요!

**다음 단계:**
- 카카오 알림톡 설정 (선택)
- 프린터 서버 설정 (로컬 PC)
- PG 결제 설정 (I'mport)

---

## 📝 참고

- 서버는 `backend/server.js` 실행
- 포트는 Railway가 자동 할당
- PostgreSQL은 Railway가 자동 생성
