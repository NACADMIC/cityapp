## 🛠️ 구현 시작 방법

### 1. 주문 취소 기능부터 시작

**백엔드 작업:**
1. `backend/database.js`에 취소 관련 메서드 추가
2. `backend/server.js`에 취소 API 추가
3. 포인트/쿠폰 복구 로직 구현

**프론트엔드 작업:**
1. 고객 앱에 취소 버튼 추가
2. POS에 취소 요청 알림 추가
3. 취소 승인/거절 UI 추가

### 2. 리뷰 작성 UI 구현

**프론트엔드 작업:**
1. 주문 완료 화면에 리뷰 작성 버튼 추가
2. 리뷰 작성 모달/화면 생성
3. 별점 UI 컴포넌트 추가
4. API 연동

### 3. SMS 알림 시스템

**백엔드 작업:**
1. SMS 서비스 선택 및 가입
2. SMS 발송 모듈 생성
3. 주문 상태 변경 시 SMS 발송 로직 추가
