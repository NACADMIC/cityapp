# 📱 SMS 알림 설정 가이드

## ✅ 구현 완료

SMS 알림 시스템이 구현되었습니다!

**기능:**
- 주문 접수 시 SMS 발송
- 배달 출발 시 SMS 발송
- 배달 완료 시 SMS 발송

---

## 🔧 SMS 서비스 선택

### 옵션 1: CoolSMS (추천)
- **가격:** 건당 약 20원
- **설정:** 간단
- **사이트:** https://www.coolsms.co.kr

### 옵션 2: 알리고
- **가격:** 건당 약 15-20원
- **설정:** 간단
- **사이트:** https://www.aligo.in

### 옵션 3: 카카오 알림톡 (무료)
- **가격:** 무료
- **설정:** 템플릿 승인 필요
- **사이트:** https://bizmessage.kakao.com

---

## ⚙️ 환경 변수 설정

### Railway 환경 변수 추가

**CoolSMS 사용 시:**
```
SMS_SERVICE=coolsms
SMS_API_KEY=your_coolsms_api_key
SMS_API_SECRET=your_coolsms_api_secret
SMS_SENDER=01012345678
```

**알리고 사용 시:**
```
SMS_SERVICE=aligo
SMS_API_KEY=your_aligo_key
SMS_API_SECRET=your_aligo_user_id
SMS_SENDER=01012345678
```

**테스트 모드 (SMS 실제 발송 안 함):**
```
SMS_TEST_MODE=true
```

## 📝 SMS 메시지 내용

### 주문 접수 알림
```
[시티반점] 주문이 접수되었습니다.
주문번호: ORD-1234567890
예상 소요시간: 30-40분
감사합니다!
```

### 배달 출발 알림
```
[시티반점] 배달이 시작되었습니다.
주문번호: ORD-1234567890
곧 도착할 예정입니다.
```

### 배달 완료 알림
```

주문번호: ORD-1234567890
맛있게 드세요! 리뷰 작성 부탁드립니다.
```

## 💰 예상 비용

**월 주문 100건 기준:**
- SMS 3회/주문 = 300건/일 = 9,000건/월
- 비용: 약 18만원/월 (건당 20원 기준)

**비용 절감 방법:**
- 카카오 알림톡 사용 (무료)
- SMS 대신 푸시 알림 사용 (PWA)
- 선택적 알림 (고객 설정)
