# 🚀 배포 실행 명령어

## Git 커밋 및 푸시

PowerShell에서 실행:

```powershell
# 현재 디렉토리 확인
cd C:\Users\j\시티반점앱

# 변경사항 확인
git status

# 모든 변경사항 추가
git add .

# 커밋
git commit -m "모든 기능 구현 완료: 주문 취소, 리뷰, 알림톡, 주소록, 재주문, 주문 수정, 쿠폰 자동 적용, 배달 예상 시간, 메뉴 품절 처리"

# 푸시
git push origin main
```

## Railway 배포 확인

1. **Railway 대시보드 접속**
   - https://railway.app
   - 프로젝트 선택

2. **배포 상태 확인**
   - Deployments 탭
   - 최신 배포 확인

3. **환경 변수 설정**
   - Variables 탭
   - 필수 변수 추가:
     ```
     DATABASE_URL (자동 생성)
     KAKAO_TEST_MODE=true
     ```

4. **로그 확인**
   - Logs 탭
   - "✅ 서버 실행 중" 메시지 확인
