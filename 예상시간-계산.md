## 💡 시간 단축 팁

### 이미 구현된 것 활용
- ✅ 즐겨찾기: 이미 일부 구현 → **1시간 절약**
- ✅ 주소록: API 있음 → **1시간 절약**
- ✅ 리뷰: API 있음 → **1시간 절약**

### 빠르게 구현하는 방법
1. **최소 기능만 먼저** (MVP)
   - 주문 취소 (기본만)
   - 리뷰 작성 (사진 제외)
   - SMS 알림 (기본만)
   - **예상: 8-10시간**

2. **점진적 개선**
   - 나중에 기능 추가
   - 사용자 피드백 반영
