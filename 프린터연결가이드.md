# 🖨️ 프린터 연결 가이드

## 📋 개요

주문이 들어오면 자동으로 주문서를 프린터로 출력하는 기능입니다.

## 🖨️ 프린터 종류

### 1. 영수증 프린터 (80mm)
- **용도**: 주문서, 영수증 출력
- **예시**: 삼성전자 RP80, 엡손 TM-T20, 스타 POS-80
- **가격**: 10만원 ~ 30만원

### 2. 레이블 프린터
- **용도**: 배달 라벨 출력
- **예시**: 바코드 라벨 프린터

## 🔌 연결 방법

### 방법 1: Node.js 프린터 라이브러리 사용

#### 1단계: 패키지 설치

```bash
cd backend
npm install node-thermal-printer
npm install node-thermal-printer-drivers
```

또는 ESC/POS 프린터용:

```bash
npm install escpos
npm install escpos-usb  # USB 연결
npm install escpos-network  # 네트워크 프린터
```

#### 2단계: 프린터 모듈 생성

`backend/printer.js` 파일 생성:

```javascript
const { ThermalPrinter, PrinterTypes, CharacterSet, BreakLine } = require('node-thermal-printer');
const { escpos } = require('escpos');

class OrderPrinter {
  constructor() {
    // USB 프린터 연결
    this.printer = null;
    this.init();
  }

  async init() {
    try {
      // 방법 1: USB 프린터
      const device = new escpos.USB();
      this.printer = new escpos.Printer(device);
      
      // 방법 2: 네트워크 프린터
      // const device = new escpos.Network('192.168.0.100', 9100);
      // this.printer = new escpos.Printer(device);
      
      console.log('✅ 프린터 연결 완료');
    } catch (error) {
      console.error('❌ 프린터 연결 실패:', error);
    }
  }

  async printOrder(orderData) {
    if (!this.printer) {
      console.error('프린터가 연결되지 않았습니다.');
      return;
    }

    try {
      const { orderId, customerName, phone, address, items, totalAmount, paymentMethod, orderType } = orderData;
      
      // 주문서 인쇄
      this.printer
        .font('a')
        .align('ct')
        .size(1, 1)
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text('      시티반점')
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text(`주문번호: ${orderId}`)
        .text(`주문시간: ${new Date().toLocaleString('ko-KR')}`)
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text(`고객명: ${customerName}`)
        .text(`전화번호: ${phone}`)
        .text(`주소: ${address}`)
        .text(`주문타입: ${orderType === 'takeout' ? '포장' : '배달'}`)
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text('주문 내역')
        .text('━━━━━━━━━━━━━━━━━━━━');

      // 메뉴 목록
      items.forEach(item => {
        this.printer
          .text(`${item.name} x ${item.quantity}`)
          .text(`  ${(item.price * item.quantity).toLocaleString()}원`);
      });

      this.printer
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text(`총 금액: ${totalAmount.toLocaleString()}원`)
        .text(`결제 방법: ${paymentMethod === 'cash' ? '현금' : '카드'}`)
        .text('━━━━━━━━━━━━━━━━━━━━')
        .text('감사합니다!')
        .text('')
        .cut()
        .close();

      console.log('✅ 주문서 인쇄 완료:', orderId);
    } catch (error) {
      console.error('❌ 인쇄 오류:', error);
    }
  }
}

module.exports = OrderPrinter;
```

## 🔧 프린터 설정

### USB 프린터
1. 프린터를 컴퓨터에 USB로 연결
2. 드라이버 설치
3. Windows: 장치 관리자에서 포트 확인 (예: COM3)
4. 코드에서 포트 번호 지정

### 네트워크 프린터
1. 프린터를 네트워크에 연결
2. IP 주소 확인 (예: 192.168.0.100)
3. 포트 번호 확인 (기본: 9100)
4. 코드에서 IP와 포트 지정

## ⚠️ 주의사항

1. **프린터 드라이버**: 프린터 제조사에서 제공하는 드라이버 설치 필요
2. **권한 문제**: Windows에서 COM 포트 접근 권한 필요
3. **네트워크 방화벽**: 네트워크 프린터 사용 시 방화벽 설정 확인
4. **용지 크기**: 80mm 영수증 용지 사용 권장
