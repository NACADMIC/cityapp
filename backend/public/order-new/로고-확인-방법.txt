━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【1단계】 파일 확인

□ backend/public/order-new/logo.jpg 파일이 있는가?
□ 파일 크기가 0이 아닌가?
□ 파일이 손상되지 않았는가?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【2단계】 브라우저 확인

1. 브라우저에서 페이지 열기
2. F12 누르기 (개발자 도구)
3. 네트워크 탭 클릭
4. 페이지 새로고침 (F5)
5. logo.jpg 파일이 로드되는지 확인
   - 200 OK → 정상
   - 404 Not Found → 파일 경로 문제
   - 다른 에러 → 확인 필요

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【3단계】 서버 재시작

로컬 서버 사용 시:
□ 서버 재시작
□ 브라우저 강력 새로고침 (Ctrl+F5)

Railway 배포 시:
□ 배포 완료 대기 (보통 2-3분)
□ 배포 완료 후 브라우저 새로고침

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【4단계】 직접 이미지 URL 확인

브라우저 주소창에 직접 입력:
  http://localhost:3000/order-new/logo.jpg
  
또는 Railway 주소:
  https://your-app.railway.app/order-new/logo.jpg

이미지가 보이면 → 경로는 정상
이미지가 안 보이면 → 파일 문제 또는 경로 문제

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 빠른 해결 방법:

1. 이미지 파일을 다시 저장
   - logo.jpg 파일을 backend/public/order-new/ 폴더에 저장
   
2. 브라우저 캐시 삭제
   - Ctrl+Shift+Delete
   - 캐시된 이미지 및 파일 선택
   - 삭제 후 새로고침

3. Railway 재배포
   - Git 푸시 후 자동 배포 대기
   - 또는 Railway 대시보드에서 수동 재배포
