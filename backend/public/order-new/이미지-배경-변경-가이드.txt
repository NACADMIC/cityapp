║     로고 이미지 배경 변경 가이드                ║
╚═══════════════════════════════════════════════╝

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

로고 이미지에서 테이블시트(placemat)를 제외한 
나머지 회색 배경을 노란색으로 바꾸는 방법:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 1】 온라인 이미지 편집 도구 사용

1. https://www.photopea.com 접속 (무료, 브라우저에서 실행)
   또는 https://www.remove.bg (배경 제거)

2. logo.jpg 파일 업로드

3. 배경 선택 도구로 회색 배경 선택
   - 테이블시트(placemat)는 제외하고 선택

4. 선택한 영역을 노란색으로 채우기
   - 색상: #FFD700 또는 #FFC107

5. 저장: logo.jpg로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 2】 Photoshop / GIMP 사용

1. 이미지 열기

2. Magic Wand Tool 또는 Quick Selection Tool로
   회색 배경 선택 (테이블시트 제외)

3. 색상 채우기: 노란색 (#FFD700)

4. 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 3】 간단한 방법 (배경 제거 후 노란 배경 추가)

1. https://www.remove.bg 접속

2. logo.jpg 업로드

3. 배경 제거 (테이블시트만 남김)

4. 노란색 배경 추가

5. 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 추천: Photopea (무료, 브라우저에서 바로 사용 가능)

1. photopea.com 접속
2. File → Open → logo.jpg
3. Magic Wand Tool로 회색 배경 선택
4. Edit → Fill → 노란색 선택
5. File → Export As → JPG → logo.jpg로 저장
