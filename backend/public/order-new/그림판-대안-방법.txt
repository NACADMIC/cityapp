━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 일반 그림판(Paint)은 투명 배경을 지원하지 않습니다!
   하지만 Windows에 기본 설치된 다른 도구들을 사용할 수 있습니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 1】 그림판 3D (Windows 10/11 기본 설치됨)

1. logo.png 파일을 우클릭 → "편집" 클릭
   → 자동으로 그림판 3D가 열림

2. 상단 메뉴에서 "Magic Select" 클릭
   (또는 상단 메뉴에서 "배경 제거" 선택)

3. 테이블시트(그릇) 부분을 드래그하여 선택
   → 자동으로 객체 인식

4. "선택 항목 복사" 클릭

5. 좌측 메뉴(☰) 클릭 → "새로 만들기" 클릭

6. 새 캔버스에 Ctrl+V로 붙여넣기
   → 배경이 자동으로 투명하게 됨!

7. 메뉴(☰) → "저장" 또는 "다른 이름으로 저장"
   → PNG 형식 선택 → logo.png로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 2】 PowerPoint (Microsoft Office 설치 시)

1. PowerPoint 실행

2. 삽입 → 그림 → 로컬 파일 → logo.png 선택

3. 그림을 클릭하면 상단에 "그림 서식" 메뉴 나타남

4. "배경 제거" 버튼 클릭
   → 자동으로 배경 제거 시도

5. 필요 시 영역 조정:
   - "유지할 영역 표시" (보라색) 클릭
   - "제거할 영역 표시" (빨간색) 클릭

6. "변경 사항 유지" 클릭

7. 그림 우클릭 → "다른 이름으로 그림 저장"
   → PNG 형식 선택 → logo.png로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 3】 Word (Microsoft Office 설치 시)

1. Word 실행

2. 삽입 → 그림 → logo.png 삽입

3. 그림 클릭 → 그림 서식 → 배경 제거

4. 자동 배경 제거 후 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 4】 그림판 3D 없을 때 - 온라인 도구

1. https://www.remove.bg 접속 (무료, 가장 쉬움)

2. logo.png 업로드

3. 자동 배경 제거

4. PNG 다운로드

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 Windows 10/11 사용자라면:
   → 그림판 3D가 기본 설치되어 있음
   → 파일 우클릭 → "편집" 하면 자동으로 열림

💡 Windows 7/8 사용자라면:
   → 그림판 3D 없음
   → remove.bg 온라인 도구 사용 (가장 쉬움)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 중요:
- PNG 형식으로 저장해야 투명 배경 유지됨
- JPG로 저장하면 투명 배경이 사라짐
- HTML에서 이미 logo.png로 설정되어 있음
