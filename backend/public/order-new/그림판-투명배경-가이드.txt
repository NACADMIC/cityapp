━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 일반 그림판(Windows Paint)은 투명 배경을 지원하지 않습니다!
   PNG 형식의 투명 배경을 만들려면 다른 도구가 필요합니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 1】 그림판 3D 사용 (Windows 10/11, 추천!)

1. logo.jpg 파일을 우클릭 → "편집" → 그림판 3D 선택
   (또는 그림판 3D 앱 직접 실행)

2. 이미지 열기

3. 상단 메뉴에서 "Magic Select" 또는 "자동 선택" 클릭

4. 테이블시트(그릇 있는 부분)를 드래그하여 선택

5. "선택 항목 복사" 클릭

6. 새 캔버스에서 "붙여넣기"

7. 배경이 자동으로 투명하게 됨

8. 메뉴(☰) → 다른 이름으로 저장 → PNG 형식으로 저장
   (중요: PNG 형식만 투명 배경 지원!)

9. 파일명: logo.png로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 2】 온라인 도구 사용 (가장 쉬움!)

1. https://www.remove.bg 접속 (한국어 지원)

2. "이미지 업로드" 클릭 → logo.jpg 선택

3. 자동으로 배경 제거 (몇 초 소요)

4. "이미지 다운로드" 클릭

5. logo.png로 저장 (자동으로 PNG 형식)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 3】 Photopea 사용 (무료, 브라우저)

1. https://www.photopea.com 접속

2. File → Open → logo.jpg 열기

3. Magic Wand Tool (단축키: W) 선택

4. 회색 배경 클릭 (테이블시트 제외하고 선택)

5. Select → Similar 선택 (비슷한 색상 모두 선택)

6. Delete 키 누르기 (배경 제거)

7. File → Export As → PNG → logo.png로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 4】 PowerPoint 사용

1. PowerPoint 실행

2. 삽입 → 그림 → logo.jpg 삽입

3. 그림 선택 → 그림 서식 → 배경 제거

4. PowerPoint가 자동으로 배경 제거

5. 그림 우클릭 → 다른 이름으로 그림 저장
   → PNG 형식으로 저장 → logo.png

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 가장 쉬운 방법: remove.bg

1. https://www.remove.bg 접속
2. logo.jpg 업로드
3. 자동 배경 제거 완료
4. PNG 다운로드
5. logo.png로 저장 후 HTML에서 src="logo.png"로 변경

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 중요 사항:

- 투명 배경은 PNG 형식만 지원
- JPG는 투명 배경 불가능
- 저장 후 HTML에서 파일명 확인:
  - logo.png → src="logo.png"
  - logo.jpg → src="logo.jpg"
