【방법 1】 remove.bg (가장 쉬움, 추천!)

1. 웹 브라우저에서 https://www.remove.bg 접속

2. "이미지 업로드" 버튼 클릭
   → logo.png 파일 선택

3. 자동으로 배경 제거 (3-5초 소요)
   → 인공지능이 자동으로 배경 인식

4. "이미지 다운로드" 버튼 클릭

5. 다운로드한 파일을 
   backend/public/order-new/ 폴더에 저장
   (파일명: logo.png)

✅ 끝! HTML에서 이미 logo.png로 설정되어 있음!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 2】 Photopea (무료, 브라우저)

1. https://www.photopea.com 접속

2. File → Open → logo.png 열기

3. 좌측 툴바에서 Magic Wand Tool 선택 (단축키: W)

4. 회색 배경 부분 클릭
   → 배경이 점선으로 선택됨

5. Select → Similar 선택
   → 비슷한 색상 모두 선택

6. Delete 키 누르기
   → 배경이 투명하게 제거됨

7. File → Export As → PNG → logo.png로 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【방법 3】 그림판 3D (Windows 10/11)

1. logo.png 파일 우클릭 → "편집" → 그림판 3D 선택

2. 상단 "Magic Select" 클릭

3. 테이블시트(그릇) 드래그하여 선택

4. "선택 항목 복사" 클릭

5. 새 캔버스에 붙여넣기

6. 메뉴(☰) → 다른 이름으로 저장 → PNG 선택

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
