# 🧪 테스트 방법 가이드

## 1️⃣ 서버 실행

```cmd
cd C:\Users\j\시티반점앱\backend
npm install
node server.js
```

## 2️⃣ PC에서 테스트

```
1. 크롬 열기
2. http://localhost:3000/order-new
3. F12 → 모바일 모드 (Ctrl+Shift+M)
4. iPhone 12 Pro 선택
```

## 3️⃣ 실제 휴대폰에서 테스트

### 같은 WiFi에 연결된 경우:

```
1. PC의 IP 주소 확인:
   cmd → ipconfig
   → IPv4 주소 확인 (예: 192.168.0.15)

2. 휴대폰 브라우저에서:
   http://192.168.0.15:3000/order-new
```

## 4️⃣ 체크리스트

### ✅ 회원가입 페이지
- [ ] 개인정보 아코디언 보임
- [ ] 클릭하면 펼쳐짐
- [ ] 모바일에서 레이아웃 정상

### ✅ 비회원 주문 페이지
- [ ] 개인정보 아코디언 보임
- [ ] 클릭하면 펼쳐짐

### ✅ 메뉴 화면
- [ ] 2칸 그리드로 표시
- [ ] 이미지가 잘림 없이 표시
- [ ] 카테고리 탭 스크롤 가능
- [ ] 가격/이름 잘 보임

### ✅ 장바구니
- [ ] 포인트 섹션 모바일 최적화
- [ ] 버튼 크기 적절

## 5️⃣ 문제 해결

### "아코디언이 안 보여요"
```
→ 브라우저 새로고침 (Ctrl+F5)
→ 캐시 삭제
```

### "레이아웃이 깨져요"
```
→ 모바일 모드 확인 (F12)
→ 화면 너비 확인
→ 새로고침
```

### "이미지가 안 나와요"
```
→ 인터넷 연결 확인 (Unsplash)
→ 서버 로그 확인
```

## 6️⃣ 개발자 모드 테스트

```
크롬 F12
→ Toggle device toolbar (Ctrl+Shift+M)
→ 해상도 변경:
  - iPhone 12 Pro (390x844)
  - Galaxy S20 (360x800)
  - iPad (768x1024)
```
