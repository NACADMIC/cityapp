# 🖼️ 메뉴 실제 사진 업로드 가이드

## 📸 방법 1: 직접 촬영 후 업로드 (가장 좋음!)

### 1단계: 음식 촬영
```
📱 스마트폰으로 음식 사진 촬영
- 밝은 조명
- 깔끔한 배경
- 정면에서 촬영
- 500x500 이상 해상도
```

### 2단계: 이미지 준비
```
1. 폴더 만들기:
   backend/public/images/menu/

2. 파일명을 영문으로 변경:
   - jjajang.jpg (짜장면)
   - jjamppong.jpg (짬뽕)
   - tangsuyuk.jpg (탕수육)
   등등...
```

### 3단계: database-simple.js 수정
```javascript
{ 
  id: 1, 
  name: '짜장면', 
  category: '오늘의메뉴', 
  price: 6000, 
  image: '/images/menu/jjajang.jpg',  // ← 이렇게!
  bestseller: 1 
}
```

## 🌐 방법 2: 무료 이미지 다운로드

### Unsplash (무료 고품질)
```
1. https://unsplash.com 접속
2. 검색: "jajangmyeon", "chinese food" 등
3. 다운로드
4. backend/public/images/menu/ 저장
5. database-simple.js 수정
```

### Pexels (무료)
```
1. https://www.pexels.com/ko-kr/ 접속
2. 음식 검색
3. 다운로드
4. 저장 후 경로 수정
```

---

## 🔗 방법 3: 온라인 이미지 URL 사용 (현재 적용!)

### Unsplash API (지금 사용 중)
```
현재 코드에 Unsplash 무료 이미지 적용됨!
서버 재시작하면 바로 보임!

장점:
✅ 무료
✅ 고품질
✅ 빠름

단점:
❌ 정확한 메뉴 사진은 아닐 수 있음
```

### Imgur 업로드
```
1. imgur.com 접속
2. 사진 업로드
3. 이미지 우클릭 → "이미지 주소 복사"
4. database-simple.js에 URL 붙여넣기

예시:
image: 'https://i.imgur.com/abc123.jpg'
```
