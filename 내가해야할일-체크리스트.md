# ✅ 내가 해야 할 일 - 체크리스트

## 🚀 배포 전 필수 작업 (내일 시제품 완성을 위해!)

### 1️⃣ Railway에 PostgreSQL 추가 ⭐ 필수!
**왜?** 데이터 영구 저장을 위해 필요합니다.

**방법:**
1. https://railway.app 접속
2. 프로젝트 선택
3. **+ New** 버튼 클릭
4. **Database** → **Add PostgreSQL** 선택
5. 완료! (자동으로 `DATABASE_URL` 설정됨)

**확인:** Railway 로그에서 "✅ PostgreSQL 데이터베이스 사용" 메시지 확인

---

### 2️⃣ I'mport 결제 설정 ⭐ 필수!
**왜?** 카드 결제를 위해 필요합니다.

**방법:**
1. https://admin.iamport.kr 접속
2. 회원가입 및 사업자 정보 등록
3. 가맹점 코드 발급 받기
4. Railway 환경 변수에 추가:
   - `IMP_KEY`: 가맹점 식별코드
   - `IMP_SECRET`: REST API Secret

**확인:** 주문 시 카드 결제 테스트

---

### 3️⃣ 코드 배포
**방법:**
```bash
# 빠른배포.bat 실행하거나
git add .
git commit -m "PostgreSQL 지원 추가"
git push
```

**확인:** Railway에서 자동 배포 완료 확인 (2-3분 소요)

---

### 4️⃣ 프린터 설정 (선택)
**왜?** 주문 접수 시 자동 출력을 위해 필요합니다.

**방법:**
- USB 프린터: Railway 환경 변수에 추가
  - `PRINTER_VENDOR_ID`: USB Vendor ID
  - `PRINTER_PRODUCT_ID`: USB Product ID
- 네트워크 프린터: Railway 환경 변수에 추가
  - `PRINTER_IP`: 프린터 IP 주소
  - `PRINTER_PORT`: 포트 번호 (기본: 9100)

**확인:** `/api/printer/test` 호출하여 테스트

---

### 5️⃣ 도메인 확인
**방법:**
1. Railway 대시보드 → Settings → Networking
2. Generate Domain 클릭 (또는 기존 도메인 확인)
3. 주소 복사

**확인:** 브라우저에서 접속 테스트

## ⚠️ 주의사항

### Railway 무료 플랜
- PostgreSQL: 월 5GB 무료 (충분함)
- 서버: 월 500시간 무료
- 유휴 시 슬립 모드 (첫 요청 시 깨어남)

- 테스트 모드와 실서비스 모드 구분

## 🎯 우선순위

1. **최우선:** Railway PostgreSQL 추가 (데이터 저장)
2. **필수:** I'mport 결제 설정 (결제 기능)
3. **권장:** 프린터 설정 (자동 출력)
4. **선택:** 도메인 커스터마이징

---

## 📞 문제 발생 시

### PostgreSQL 연결 안 됨
- Railway에서 PostgreSQL 서비스 확인
- `DATABASE_URL` 환경 변수 확인

### 결제 안 됨
- `IMP_KEY`, `IMP_SECRET` 확인
- I'mport 대시보드에서 결제 내역 확인

### 프린터 안 됨
- 프린터 라이브러리 설치 확인
- 환경 변수 확인
