# 🖨️ POS PC 프린터 서버 (간단 버전)

---

## 📦 설치 방법 (2단계)

### 1단계: POS PC로 파일 복사
`프린터서버-독립버전` 폴더 전체를 USB로 복사

### 2단계: 실행
1. **Node.js 설치** (최초 1회만)
   - https://nodejs.org 접속 → LTS 다운로드 → 설치

2. **`한번에-설치-시작.bat` 더블클릭**
   - 자동으로 설치하고 서버 시작
   - 창을 닫지 말고 열어둠

---

## ⚙️ Railway 설정

### 1. POS PC IP 주소 확인
명령 프롬프트에서:
```
ipconfig
```
→ "IPv4 주소" 확인 (예: 192.168.0.100)

### 2. Railway 환경 변수 설정
Railway 대시보드 → Variables:
- 이름: `PRINTER_SERVER_URL`
- 값: `http://[확인한 IP]:3001`

**예시:**
```
http://192.168.0.100:3001
```

⚠️ **같은 네트워크에 있어야 접근 가능!**  
다른 네트워크면 `프린터터널시작.bat` 사용 (Cloudflare Tunnel)
