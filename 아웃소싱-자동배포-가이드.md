## 📋 3단계로 완료

### 1단계: GitHub에 코드 업로드

#### Git 설치 (아직 안 했다면)
1. https://git-scm.com/download/win 다운로드
2. 설치 (기본 설정으로 진행)
3. 설치 후 **컴퓨터 재시작**

#### GitHub에 업로드
```bash
# 1. GitHub에서 새 저장소 생성
#    https://github.com/new
#    저장소 이름: city-restaurant-app

# 2. 프로젝트 폴더에서 실행
git init
git add .
git commit -m "시티반점 앱 초기 업로드"
git branch -M main
git remote add origin https://github.com/당신계정/city-restaurant-app.git
git push -u origin main
```

---

### 2단계: Railway 배포 (5분 소요)

1. **Railway 접속**
   - https://railway.app
   - **GitHub로 로그인**

2. **프로젝트 생성**
   - **New Project** 클릭
   - **Deploy from GitHub repo** 선택
   - 방금 만든 저장소 선택
   - **Deploy** 클릭

3. **자동 배포 대기** (3-5분)
   - Railway가 자동으로:
     - Node.js 설치
     - 의존성 설치
     - 서버 시작
     - 데이터베이스 생성

4. **도메인 받기**
   - 배포 완료 후 프로젝트 클릭
   - **Settings** → **Networking**
   - **Generate Domain** 클릭
   - 주소 복사! (예: `cityapp-production.up.railway.app`)
